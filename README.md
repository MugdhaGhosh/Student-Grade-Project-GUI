🎓 Student Grade Calculator System
The Student Grade Management System is a Java-based desktop application developed using Object-Oriented Programming (OOP) concepts and Java Swing. The system helps students calculate and manage their academic performance by processing semester-wise course information and generating GPA and CGPA results automatically.

The application includes a secure login and registration system, allowing users to access the CGPA calculation module after successful authentication. It simplifies academic result calculation by reducing manual effort and minimizing calculation errors.

📚 OOP Concepts Implemented
Encapsulation

User information and academic data are stored within classes and accessed through methods.

Classes and Objects

The project is built using classes such as:

User
LoginSystem
GradeCalculator
Semester
Inheritance

Inheritance can be extended for future academic modules and user roles.

Abstraction

Grade calculation and result generation are separated into dedicated methods.

Polymorphism

The project structure supports future implementation of method overriding and overloading.

🏗 System Workflow
User registers a new account.
User logs into the system.
The system verifies login credentials.
User enters student information.
User enters semester and course details.
The system calculates grades and grade points.
Semester GPA is generated.
Overall CGPA is calculated.
Final result, division, and pass/fail status are displayed.

▶️ How to Run
Clone the repository.
Open the project in your preferred Java IDE.
Compile all Java files.
Run the LoginSystemGUI class.
Register a new account or log in.
Start entering academic information and calculate CGPA.

🔒 Limitations

No database integration.
Limited user profile management.
Console version has a basic user interface.
No data export functionality.
